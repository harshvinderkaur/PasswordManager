This Password Manager is a simple desktop application built using Python and Tkinter. It allows users to securely generate, store and retrieve passwords for websites and applications. The app offers a graphical interface to manage passwords and utilizes JSON for saving and retrieving data.

Features:

Passowrd Search:  Users can search for saved passwords by entering a website or app name. Upon searching, a pop-up displays the associated email and password.
![image](https://github.com/user-attachments/assets/018de22a-0a42-480a-af6d-ac36f487518e)

Add Passwords: Users can add new entries by providing a website/app name, email, and password. If you don't have a password in mind, the app can generate one for you by clicking the Generate button. A pop-up will appear, allowing you to customize the number of characters, numbers, and symbols in the password.
